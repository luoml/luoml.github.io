ie无法跳转到自定义404页面

发表于 2016-06-24  |  2016-06-24   |   分类于 疑难杂症   |     |   阅读次数

今天遇到一个问题：自定义404页面在IE浏览器无法显示，但其他浏览器都可以正常显示。
原来自定义的404页面大小必须大于512字节，否则IE会使用其自带的404页面替代你的自定义页面。

chrome无法打开任何页面

发表于 2016-06-24  |  2016-10-13   |   分类于 疑难杂症   |     |   阅读次数

Chrome莫名其妙地崩溃了，连设置页面也无法打开，总是提示喔唷 崩溃啦。
用IE、Firefox等浏览器都可正常上网。
临时解决办法：在Chrome快捷方式中增加：--no-sandbox。

今天，终于彻底解决。
步骤：1.卸载； 2.通过CCleaner卸载注册表；3.重装Chrome
