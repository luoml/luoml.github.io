如何在真机上调试Android应用程序

发表于 2016-09-05   |   分类于 android   |     |   阅读次数

在手机上启用调试模式

设置 - 开发人员选项 - 调试, 勾选USB调试。

连接数据线至电脑，安装驱动

如果是华为手机，将USB连接方式改为PC助手(HiSuite)，就会自动安装驱动了。

检测设备是否已装载

开始 - 运行 - adb devices，如下，表示设备已正常装载。

注意，需要先将${android-sdk}\platform-tools加入path环境变量，否则会提示adb命令找不到。

配置Eclipse执行Tagret

Run as - Run Configurations - Target, 勾选Always prompt to pick device。

选择Android设备


注意，如果设备为Offline状态，可以通过Windows - Open perspective - Other - DDMS - Reset adb解决。

OK，大功告成！选择Android设备，即可在真机上进行开发测试了。
